# Hover Translator

Instant word translations on hover — Chrome extension.

**Freemium:** 100 words/day free · **Premium:** Unlimited, €14.99 one-time

[![Install](https://img.shields.io/badge/Chrome-Install-blue)](https://chrome.google.com/webstore)
[![Privacy Policy](https://img.shields.io/badge/Privacy-Policy-green)](https://privacy-policy.html)

## Features
- Translates any word on any webpage on hover
- Context-aware translations (reads the surrounding sentence)
- 30+ languages
- Proper noun detection with Wikipedia descriptions
- German separable verb recognition
- Works fully offline after install (no account needed)

## Privacy
No servers. No tracking. All data stays on your device.
[Privacy Policy](https://dani3c.github.io/hover-translator/privacy-policy.html)
